  Creating Digital Graphics  
     
 
In the Creating Digital Graphics unit we looked at two graphic images one raster and one vector. We looked at their quality and if they were fit for purpose. We also looked at if this could be improved and if so what the problem with the graphic was. We also were set a task of creating a flyer for an outdoors music festival which was not an assignment but was preparing us for the assignment to come. As our task was to develop a design for a new handheld console.

We were told to come up with a quickstart user guide that would be a vector graphic that could possibly be printed so needed to be quite high DPI so I went with 300dpi that should be good enough. The other was an advert that could be displayed on the web or possibly printed. I developed two versions of this one for the web at quite low resolution and in PNG format. The other was to be printed so I went with 300 dpi like the vector graphic that was in TIFF format that is better for printing. With the vector graphic user guide it was decided that it would be in PDF format that would make it easy for printing.

The user guide was done in Adobe Illustrator and was all my own work.

The advert was built using readymade assets namely images and edited to cater for my needs in the design of handheld console. As there are many handheld consoles out there, there was a plentiful supply of suitable images.

I find that I'm quite good at manipulating graphics. So I built the different designs from the images I acquired from the internet namely the web and google images search.

After completing the design I asked other students for feedback and based on the feedback I made modifications to the final design. I have put a selection of images below and a description of what stage they are at
